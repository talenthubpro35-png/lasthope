## Getting Started

### Accessing the Platform

1. Open your web browser and go to: **http://localhost:5000**
2. You'll see the TalentHub Pro landing page
3. Click **"Get Started"** or navigate to the login page

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- JavaScript enabled

## Login & Authentication

### Creating an Account

1. Click **"Sign Up"** on the login page
2. Select your role:
   - **Candidate** - If you're job hunting
   - **Recruiter** - If you're hiring
   - **Admin** - Platform administrator (invitation only)
3. Enter your credentials:
   - Email address
   - Password (minimum 8 characters, should be strong)
   - Confirm password
4. Click **"Create Account"**

### Logging In

1. Enter your **email** and **password**
2. Click **"Login"**
3. You'll be redirected to your personalized dashboard

### Password Management

- Click **"Forgot Password?"** on the login page to reset your password
- Check your email for the password reset link
- Follow the instructions to create a new password

## Troubleshooting

### "Can't Log In"

**Problem:** Incorrect password or email not found

**Solutions:**
- Check email spelling
- Use "Forgot Password" to reset
- Clear browser cookies and try again
- Use different browser if issue persists

### "Can't Upload Resume"

**Problem:** File format not accepted

**Solutions:**
- Use PDF or Word (.docx) format only
- File size must be under 5MB
- Check file isn't corrupted
- Try different file if issue persists

### "Application Not Submitting"

**Problem:** Form shows errors

**Solutions:**
- Fill all required fields (marked with *)
- Check internet connection
- Clear browser cache
- Try different browser
- Contact support if problem persists

### "Can't See Job Applications"

**Problem:** Applications list is empty

**Solutions:**
- Wait 24 hours - applications may take time to appear
- Check your spam/junk email for confirmation
- Verify you're logged in as the correct user
- Refresh the page
- Check if job is still active

### "CV Builder Not Loading"

**Problem:** Page shows blank or error

**Solutions:**
- Clear browser cache
- Refresh the page
- Try different browser
- Log out and log back in
- Disable browser extensions

### "Slow Performance"

**Problem:** Website loading slowly

**Solutions:**
- Check internet connection
- Clear browser cache and cookies
- Disable browser extensions
- Close other tabs/applications
- Use modern browser (Chrome, Firefox, Safari, Edge)
